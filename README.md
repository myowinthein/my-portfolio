# Project Setup
```shell
-- clone the repository 
git clone git@github.com:myowinthein/personal.git

-- install node packages
npm install or yarn
```

# Run the Project
```shell
-- start development server (for local)
npm run dev or yarn dev

-- build & preview (for local)
npm run build or yarn build
npm run start or yarn start

-- open in browser
http://localhost:4000
```
